# AfghanTours — Site Images Instructions (Strict Enforcement)

## Purpose
This file controls image handling across the entire site.

These rules apply whenever Copilot creates or updates any visual section.

---

## Image Philosophy
Images must feel:
- premium
- authentic
- place-specific
- emotionally strong
- operationally credible
- visually coherent

---

## Non-Negotiable Rules
- Do not use low-quality imagery.
- Do not leave broken image references.
- Do not mix wildly inconsistent aspect ratios inside a single grid or section.
- Do not use heavy decorative filters unless explicitly requested.

---

## File Structure
Use predictable asset structure under `/public/assets/images/`.

Preferred folders:
- heroes
- tours
- attractions
- hubs
- hotels
- dishes
- about
- maps
- placeholders

---

## Naming Rules
Use lowercase, hyphenated, descriptive names only.

Good:
- kabul-hero.webp
- band-e-amir-lake-1.webp
- herat-citadel-dusk.webp

Bad:
- IMG_4402.webp
- New Hero Final.webp
- image1.jpg

---

## Preferred Formats
- `.webp` for primary photography
- `.svg` only for vector-based assets like maps or icons

---

## Size Guidance
Hero:
- ~2400px wide preferred

Card:
- ~1200px wide preferred

Detail-page:
- ~1600px wide preferred

---

## Treatment Rules
Allowed:
- subtle dark overlays for text legibility
- tasteful cropping
- light contrast normalization

Avoid:
- dramatic stylized filters
- overly cinematic color grades that reduce realism
- cluttered collages unless explicitly designed that way

---

## Section Rules

### Hero Sections
Use one strong image or a carefully curated gallery.
The image must support the page identity immediately.

### Tour Pages
Prioritize:
- destination mood
- route logic support
- accommodation or experience context where needed

### Attraction Pages
Prioritize:
- the attraction itself
- recognizable context
- supporting visuals only if useful

### Hub Pages
Prioritize:
- city entry / skyline / landmark / road / airport context

### Hotel Sections
Use:
- exterior first
- then pool / lobby / room if needed

### Dishes Pages
Use:
- authentic food photography
- warm natural tones
- clean compositions

---

## Fallback Rules
If a specific image is unavailable:
- use a relevant category or location placeholder
- do not leave empty or broken areas
